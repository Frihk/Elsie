# Elsie Backend

Go API for the Elsie project using the standard `net/http` package.

## Run

```sh
go run .
```

## Routes

- `POST /api/login`
- `GET /api/content`
- `POST /api/content`
- `GET /api/settings`
- `POST /api/settings`
- `POST /api/upload`

## Environment

Copy `.env.example` to `.env` for local values. The real `.env` file is ignored by Git.
